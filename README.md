<h1> SCP  </h1> - <b> To Copy File to Pi </b>
    
    scp file1.txt file2.txt pi@192.168.1.xx:folder1/


To Copy All File from a Dir to a Pi
    
    scp -r /Users/name/3 pi@192.168.0.xx:/home/pi/foldername
